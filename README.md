# Arduino_LCD_Calculator
I used an LCD and an Arduino UNO to create a calculator. It can add/subtract/multiply and divide 2 numbers. The user inputs the numbers using the Serial Monitor.
Fist thing do the wirings.
Seconly run the Find_I2C_Address.ino progam to get the right address of your LC display, and add it to the code.
And finally run the LCD_Calculator.ino program and now you have a running calculator!
